# Skill: Frontend React (TypeScript + Vite)

## Ne Zaman Yükle
- Yeni UI bileşeni eklerken
- Dashboard değişikliğinde
- Form veya state yönetimi gerektiğinde

## Bu Skill Ne Bilir

### Teknoloji Stack
- React 18 (Functional Components)
- TypeScript (Strict typing)
- TanStack Query (Server state)
- Zustand (Client state)
- Tailwind CSS (Styling)
- Lucide React (Icons)

### Standartlar
- Componentler `frontend/src/components/` altında olmalı.
- API çağrıları için `frontend/src/api/` (Axios) kullan.
- Formlar için `react-hook-form` + `zod` ikilisini kullan.
- Responsive tasarım ve dark mode uyumuna dikkat et.

## Yapma
- Inline style kullanma (Tailwind kullan).
- `any` tipi kullanma.
- Karmaşık logic'leri component dışına (hooks/services) taşı.
